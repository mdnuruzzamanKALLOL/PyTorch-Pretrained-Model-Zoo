Files
-----
  load_pretrained.py    — load ImageNet weights, replace head
  feature_extraction.py — freeze backbone, train head only
  fine_tuning.py        — dual-LR AdamW (backbone=1e-5, head=1e-3)

Requirements
------------
  pip install torch torchvision

Steps
-----
  1. Set NUM_CLASSES in each script
  2. python load_pretrained.py       <- verify weights load
  3. python feature_extraction.py    <- fast baseline
     OR
     python fine_tuning.py           <- higher accuracy

Pretrained weights
------------------
  Swin_V2_B_Weights.IMAGENET1K_V1
  Top-1 ImageNet : 84.6%

Head to replace
---------------
  model.head = nn.Linear(model.head.in_features, NUM_CLASSES)
  in_features = 1024  (embed_dim=128, 8x after 3 PatchMerging)
