How to Run — MobileNet V3 Large Python Scripts
==============================================

1. Prepare dataset in ImageFolder format:
   data/
     train/  class1/  class2/ ...
     val/    class1/  class2/ ...

2. Edit NUM_CLASSES and DATA_DIR in train.py.

3. Train the model:
   python train.py

4. Run inference:
   python inference.py your_image.jpg

Files:
  mobilenet_v3_large.py  — Full V3 Large architecture (SE + HardSwish + HardSigmoid)
  train.py    — Training with CosineAnnealingLR
  inference.py — Top-K prediction with visualization
