Files
-----
  load_pretrained.py    — load ImageNet weights, replace model.fc
  feature_extraction.py — freeze backbone, train fc only
  fine_tuning.py        — dual-LR AdamW (backbone=1e-5, fc=1e-3)

Requirements
------------
  pip install torch torchvision

Steps
-----
  1. Set NUM_CLASSES in each script
  2. python load_pretrained.py       <- verify weights load
  3. python feature_extraction.py    <- fast baseline
     OR python fine_tuning.py        <- higher accuracy

Head to replace
---------------
  model.fc = nn.Linear(model.fc.in_features, NUM_CLASSES)
  in_features = 1024

Pretrained weights
------------------
  ShuffleNet_V2_X1_0_Weights.IMAGENET1K_V1
  Top-1 ImageNet : 69.4%
