Files
-----
  regnet_x_32gf.py   — SE + XBlock + RegNet + all factory functions
  train.py      — Adam + StepLR training loop
  inference.py  — top-K single-image prediction

Requirements
------------
  pip install torch torchvision pillow matplotlib scikit-learn

Steps
-----
  1. Set NUM_CLASSES in train.py
  2. python train.py
     -> saves regnet_x_32gf_best.pth
  3. python inference.py your_image.jpg

Config
------
  Series        : RegNet X
  depths        : [2, 7, 13, 1]
  widths        : [336, 672, 1344, 2520]
  group_widths  : [168, 168, 168, 168]
  FC in_features: 2520
  Parameters    : ~107.8M
  Batch size    : 8
  Epochs        : 20
  Optimizer     : Adam (lr=0.001)
  Scheduler     : StepLR (step_size=7, gamma=0.1)
  Input size    : 224x224
