How to Run — MNASNet 1.3 Using Weight File
==========================================

1. load_pretrained.py
   Loads torchvision MNASNet 1.3; replaces classifier[1].
   python load_pretrained.py

2. feature_extraction.py
   Freezes backbone; trains only classifier[1].
   python feature_extraction.py

3. fine_tuning.py
   Dual LR: backbone=1e-5, classifier[1]=1e-3 (AdamW + CosineAnnealing).
   python fine_tuning.py

Notes:
  - Classifier structure: Sequential([Dropout, Linear(1280->classes)]).
    Replace classifier[1] for transfer learning.
  - in_features = 1280 for all MNASNet alpha variants.
  - MNASNet 1.3 has no official pretrained weights in torchvision.
load_pretrained.py initializes with random weights.
