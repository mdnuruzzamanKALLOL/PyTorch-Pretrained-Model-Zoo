Files
-----
  load_pretrained.py    — load ImageNet weights, replace fc
  feature_extraction.py — freeze backbone, train only fc
  fine_tuning.py        — dual-LR AdamW (backbone=1e-5, fc=1e-3)

Requirements
------------
  pip install torch torchvision

Data Layout
-----------
  data/
    train/<class_name>/image.jpg ...
    val/<class_name>/image.jpg ...

Steps
-----
  1. Set NUM_CLASSES to match your dataset in each script
  2. python load_pretrained.py       <- verify weights load
  3. python feature_extraction.py    <- fast, good baseline
     OR
     python fine_tuning.py           <- higher accuracy

Pretrained weights used
-----------------------
  Wide_ResNet101_2_Weights.IMAGENET1K_V2
  Top-1 ImageNet accuracy : 82.5%

Classifier head to replace
--------------------------
  model.fc = nn.Linear(2048, NUM_CLASSES)
  in_features = 2048  (Bottleneck.expansion=4, planes=512)
