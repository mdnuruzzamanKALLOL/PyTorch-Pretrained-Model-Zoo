EfficientNet-B2 — Python Scripts
========================================

FOLDER STRUCTURE
----------------
  efficientnet_b2.py   ← model architecture (from scratch)
  train.py               ← training script
  inference.py           ← single-image inference
  How to run.txt         ← this file

REQUIREMENTS
------------
  pip install torch torchvision scikit-learn pillow matplotlib

DATASET FORMAT
--------------
  data/
    train/
      class_a/  image1.jpg  image2.jpg ...
      class_b/  ...
    val/
      class_a/  ...
      class_b/  ...

TRAINING
--------
  python train.py

  Key settings in train.py:
    DATA_DIR    = './data'
    NUM_CLASSES = 10         ← change to your number of classes
    BATCH_SIZE  = 32
    EPOCHS      = 30
    LR          = 1e-3

  Saves: efficientnet_b2_best.pth

INFERENCE
---------
  python inference.py <image_path> [class_names.json] [weights.pth] [num_classes]

  Examples:
    python inference.py photo.jpg
    python inference.py photo.jpg classes.json efficientnet_b2_best.pth 10

  class_names.json format:
    ["cat", "dog", "bird", ...]

INPUT RESOLUTION
----------------
  EfficientNet-B2 expects 260×260 images.
  The scripts resize to 292 then crop to 260×260 automatically.
