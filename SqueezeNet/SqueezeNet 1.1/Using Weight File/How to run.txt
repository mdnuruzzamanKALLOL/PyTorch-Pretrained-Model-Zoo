Files
-----
  load_pretrained.py    — load ImageNet weights, replace classifier[1]
  feature_extraction.py — freeze backbone, train classifier[1] (Conv2d)
  fine_tuning.py        — dual-LR AdamW (backbone=1e-5, head=1e-3)

Requirements
------------
  pip install torch torchvision

IMPORTANT — Classifier is Conv2d, not Linear
---------------------------------------------
  model.classifier[1] = nn.Conv2d(512, NUM_CLASSES, kernel_size=1)
  in_channels = 512  (same for all SqueezeNet variants)

Steps
-----
  1. Set NUM_CLASSES in each script
  2. python load_pretrained.py       <- verify weights load
  3. python feature_extraction.py    <- fast baseline
     OR python fine_tuning.py        <- higher accuracy

Pretrained weights
------------------
  SqueezeNet1_1_Weights.IMAGENET1K_V1
  Top-1 ImageNet : 58.2%
