Files
-----
  squeezenet1_1.py   — model architecture (Fire module + SqueezeNet class)
  train.py      — training loop with StepLR scheduler
  inference.py  — top-K single-image prediction

Requirements
------------
  pip install torch torchvision pillow matplotlib scikit-learn

Data Layout
-----------
  data/
    train/<class_name>/image.jpg ...
    val/<class_name>/image.jpg ...

Steps
-----
  1. Set NUM_CLASSES in train.py
  2. python train.py
     -> saves squeezenet1_1_best.pth
  3. python inference.py your_image.jpg

Training config
---------------
  Batch size  : 128
  Epochs      : 20
  Optimizer   : Adam (lr=0.001)
  Scheduler   : StepLR (step_size=7, gamma=0.1)
  Loss        : CrossEntropyLoss
  Input size  : 224x224
  Parameters  : ~1.24M
